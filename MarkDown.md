### Aprendendo a usar a Linguagem Markdown

## Negrito
__Negrito__ ou **Negrito** é o usando '##' oou '__'

## Italico
_Negrito_ ou *Negrito* é o usando '#' oou '_'

## Riscado
~~Negrito~~ é o usando '~~' 

## Titulo 
a quantidade de '#' define a quantidade de nível do titulo
### title
## title
# title

## Lista Enumerada
ele vai pega ordem independente do numero posto antes do ponto
1. teste
1. teste
   1. subitem em romano
   2. subitem em romano
33. tesste

## Lista demarcada
usando '*' ou '-' faz-se topicos
- teste
* subitem em romano
* subitem em romano
- tesste

## Lista de Tarefas:
usando - [ ] faz-se tópicos
quando marcado - [x]
- [ ] teste1
- [x] teste2
- [ ] teste3

## Comando
usando '`' entre palavras para linhas

usando '```' entre mensagens para trecho de codigo 

o comando `prinft` serve, no c, para expor uma mensagem na tela

o codigo abaixo serve para printar hello world na tela

```
# include <stdio.h>
int main() {
  printf("hello world\n");
  return 0;
}
```
